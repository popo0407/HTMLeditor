HTMLエディタ読み込みサンプル

📝 このHTMLファイルについて

このHTMLファイルは、TinyMCEエディタの読み込み機能で対応しているすべての要素を含んでいます。以下の要素が含まれています：

  • 見出し（h1, h2, h3）
  • 段落とテキスト
  • リスト（番号付き・番号なし）
  • テーブル
  • 強調表示（太字・斜体）
  • 引用
  • コードブロック
  • ボタン
  • 色付きボックス

🎯 基本的なテキスト要素

これは通常の段落です。TinyMCEエディタでは、このような基本的なテキストも適切に読み込まれます。

太字のテキストと斜体のテキストも正しく表示されます。

また、ハイライト表示や色付きテキストも対応しています。

📋 リスト要素

番号なしリスト

  • リスト項目1
  • リスト項目2
  • リスト項目3
    • ネストしたリスト項目1
    • ネストしたリスト項目2

番号付きリスト

  1. 最初の項目
  2. 二番目の項目
  3. 三番目の項目

🎯 使用方法

  1. このHTMLファイルをブラウザで開く
  2. すべての内容を選択（Ctrl+A）
  3. コピー（Ctrl+C）
  4. TinyMCEエディタの読み込みテキストエリアに貼り付け
  5. 「読み込み」ボタンをクリック
